YNAL: You Need A License

I made this site because I was tired of having to google "MIT License" all the time. This is a curlable server to add a license to a project. It reads the Accept header and responds with plaintext, HTML, or JSON appropriately. Try it out with:

curl -s --output LICENSE.txt https://ynal.packrat386.com/mit

This site should not be considered any kind of authority on the validity of these licenses. Do you own research and all that jazz.
